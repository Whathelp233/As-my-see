# As-my-see 知识库

一个结构化的个人技术知识库，包含机器人学、Qt、Linux、C++、ROS 等技术领域的文档和笔记。

## 📁 目录结构

```
As-my-see/
├── docs/                    # 技术文档
│   ├── cpp/                # C++ 相关
│   ├── qt/                 # Qt 框架
│   ├── linux/              # Linux 系统
│   ├── robotics/           # 机器人学
│   ├── ros/                # ROS 系统
│   ├── vision/             # 计算机视觉
│   ├── embedded/           # 嵌入式系统
│   └── general/            # 通用技术
├── notes/                  # 学习笔记
│   ├── signals/            # 信号处理
│   ├── learning/           # 学习方法
│   ├── math/               # 数学基础
│   └── electronics/        # 电子技术
├── projects/               # 项目记录
├── resources/              # 学习资源
├── knowledge_expansions/   # 知识扩展
└── tools/                  # 工具脚本
```

## 🚀 快速开始

### 浏览文档
1. 进入对应技术目录：`cd docs/[技术领域]`
2. 查看 Markdown 文档
3. 使用知识扩展获取最新资源

### 使用工具
```bash
# 文档分析
./tools/simple_analyzer.sh

# 文档分类
./tools/document_classifier.sh

# 内容扩展
./tools/content_expander.sh
```

## 代码示例

```bash
#!/bin/bash
# Linux Shell脚本示例

echo "系统信息:"
echo "主机名: $(hostname)"
echo "内核版本: $(uname -r)"
echo "系统时间: $(date)"

# 文件操作
echo "当前目录文件:"
ls -la

# 进程检查
echo "运行中的进程:"
ps aux | head -10

echo "脚本执行完成"
```

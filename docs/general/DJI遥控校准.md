# DJI遥控校准
> 更新时间: 2026年02月27日

# method

1. [安装遥控器驱动](https://www.robomaster.com/zh-CN/products/components/detail/122)

2. [安装DT7 & DR16 调参软件](https://www.dji.com/cn/dt7-dr16-rc-system/download)

3. 开启遥控器，使用Micro-USB连接遥控和电脑
4. 打开调参软件，点击校准，遥控会B--B--，按照提示完成即可

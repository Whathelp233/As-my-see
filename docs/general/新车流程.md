# 新车流程
> 更新时间: 2026年02月27日

新车流程：
1.出urdf方便跑仿真
2.扒主仓代码，catkin clean环境，catkin build环境
3.对应电机改配置文件id
4.手动跑程序查询rviz电机旋转方向与实际是否一致，位姿是否正确，用plotjuggler查看imu数据是否正常
5.部署自启（具体看队内文档），确保开控无问题
6.调pid，基础的是yaw，pitch，trigger，其他看车实际情况调
7.调自瞄
8.（开发每一个部分都需要从master上开新分支，改完pr）

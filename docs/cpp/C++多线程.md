# 1. 程序并行执行的条件

1. 硬件：多个处理器
2. 软件：（程序编写上）
   - 多进程（多个进程并行）
   - 多线程（同一个进程分成多个线程并行）

对于多线程，主要关注的是线程间的同步措施，用于确保线程安全；

对于多进程，主要关注的是进程间的通信机制，用于进程间传递消息和数据；

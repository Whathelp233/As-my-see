# RadioButton

> 单选按钮

通过`ui->XXX->setChecked(true)`来设置默认



# CheckButton

> 多选按钮

除了继承的信号外，还有一个自己的信号stateChanged，2是选中，0是未选中，1是半选(属性qCheckBox的tristate选中)
